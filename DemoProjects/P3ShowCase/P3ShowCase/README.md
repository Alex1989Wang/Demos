#  README

### Background

在进行图片编辑之前，将广色域图片（Display P3色域）转换为sRGB的色域，丢失了广色域的色彩。

项目[Wide Color Viewer](https://github.com/edmonston/wide-color-viewer)中可以看到在日常场景中，镜头可以捕获更广的色域。而颜色色值超过sRGB的事物也是无处不在。
